# 規格：警部牌（Inspector）

狀態：**已批准（2026-06-01）** ← 4 個決策點全採用提案值，進入實作
關聯研究：繁體中文版特殊卡，來源見 CLAUDE.md / 爬蟲結果。

## 1. 目標與動機
加入「警部」牌，提供一種**有別於偵探的抓犯人方式**：偵探是「當下指認、立刻分勝負」；警部是**「先公開押注、結算到遊戲結束才看中沒中」**，玩法更有張力。

## 2. 行為規格（提案值，待確認）
- 牌型代號：`inspector`，顯示名「警部」。
- **公開持有**：拿到警部的人，這張牌要**讓所有人看得到**（畫面顯示「👮 某人公開持有警部」）。
- **出牌條件**：和偵探一樣，**手牌 ≤3 張**才能打出。
- **打出時**：指定一名玩家，把警部**放到那位玩家面前**（畫面標記「👮 警部盯著 ○○○」），**當下不結算**；警部牌離開手牌、置於桌面。
- **結算時機（遊戲結束時）**：遊戲一旦結束，檢查被警部盯著的玩家**當下手上是否持有犯人牌**：
  - 是 → **警部玩家獲勝**，比照神犬計 **+3**，犯人陣營 0，其他平民 +1。
  - 否 → 警部沒中，不影響原本的結局。

## 3. 需要你拍板的決策點 ⚠️（SDD 就是要先把這些定下來）
1. **牌庫怎麼放**：4 人局固定 16 張。提案：**加入警部×1，並移除 1 張「一般人」**以維持 16 張。（或：替換掉神犬？請選）
2. **與其他結局衝突**：
   - 若**犯人逃脫**（打出最後一張犯人牌），此時犯人牌已被打出、不在手上 → 依規格警部「沒中」。**接受嗎？**
   - 若**偵探或神犬已抓到犯人**而結束 → 那位犯人此刻仍持有犯人牌 → 警部也算中。這時**警部和偵探/神犬同時得分嗎**？提案：**可同時各自得分**（兩個都贏）。接受嗎？
3. **和局（牌都打完）**：若結束時根本沒人持有犯人牌（已被打出/棄掉）→ 警部沒中。接受嗎？
4. **公開資訊在本機同桌版**怎麼呈現：提案在畫面上方常駐顯示「👮 警部在 ○○○ 手上 / 警部盯著 ○○○」。

## 4. 介面/流程
- 手牌中警部牌：手牌≤3 時可點，否則灰掉並提示「手牌需≤3張」。
- 打出 → 進入選目標（沿用現有 target 流程）→ 設定「警部盯著 X」的公開標記。
- 遊戲結束畫面：若警部命中，結算文字加一行「👮 警部 ○○○ 指認成功！」並計分。

## 5. 不做（Out of scope）
- 本次只做警部，不做多人(3–8人)規則、不動其他卡。
- 線上版的警部呈現等之後做線上本體再處理。

## 6. 驗收條件（第⑤步要逐項測；對應 criminal-dance-debug）
- A. 牌庫含警部×1、總數仍 16、無壞牌。
- B. 手牌 4 張時警部不可打（提示≤3）；3 張時可打。
- C. 打出後警部離手、出現「警部盯著 X」公開標記、當下不結算。
- D. 結束時 X 持有犯人 → 警部 +3、犯人 0、平民 +1。
- E. 犯人逃脫結束 → 警部不得分（犯人牌已不在手上）。
- F. 偵探抓到犯人且警部也盯對人 → 兩者皆得分（依決策2提案）。
- G. 公開標記在本機同桌畫面正確顯示。

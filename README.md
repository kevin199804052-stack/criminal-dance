# 犯人在跳舞（Criminal Dance）網頁版

把桌遊「犯人在跳舞 / 犯人は踊る」做成的網頁版，純 HTML + JavaScript，單檔即可遊玩。

## 怎麼玩
直接用瀏覽器打開 [`criminal-dance.html`](criminal-dance.html) 即可，不需安裝任何東西。

兩種模式：
- **本機同桌**：大家共用一台裝置輪流玩，靠「換人遮擋畫面」保護手牌。
- **線上連線**（開發中）：每人各自裝置，用房號加入同一局；後端使用 Firebase Realtime Database。

## 規則重點
- 標準版、固定 4 人，使用 16 張牌、每人發 4 張。
- 目標：偵探／神犬找出犯人，犯人設法把犯人牌留到最後一張打出逃脫。
- 12 種卡片：第一發現者、犯人、偵探、不在場證明、共犯、目擊者、情報交換、謠言、交易、少年、神犬、一般人。
- 計分：偵探破案 +2、神犬立功 +3、犯人逃脫犯人與共犯各 +2、平民 +1；累積總分先達 `WIN_SCORE`（預設 10）且唯一最高者贏得整場。

## 技術
- 前端：純 HTML / CSS / JavaScript，無編譯步驟。
- 連線：Firebase Realtime Database（compat SDK，CDN 載入）。

> 注意：Firebase 資料庫目前為測試模式，正式開放遊玩前需設定安全規則。
